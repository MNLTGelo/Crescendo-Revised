Crypto Crash

Crypto Crash

Introduction
Crypto Crash is an exciting game of rocket where player predict how high up the rocket will travel before it explodes. Correct pediction before it explodes will be victorious and incorrect prediction will lose.


How to Play
1. At the start of the draw, player to place a bet.
2. When the timer runs out , the rocket will move with payout indicator.
3. If player cashout before it explodes, player win with the same payout indicator value when cashout.
If player cashout when it explode, player lose.


Bet Types

Bet Bet on the game round.

** Player can only place a bet on a round game.

Example 1 :

Player bet $1.-

Cashout Result : x5.2

Player win 5.2

Bet 1 - 50,000

Crypto Roulette

Introduction
Roulette is one of the popular online games with numbers 0 – 36


How to Play
1. At the start of the draw, place bet on your selected bet types.
2. When the timer runs out the system will randomly select Result from 0-36
3. If roulette results match with your bet, you win or else lose.


Bet Types

Black, Red Bet on Black, Red or Green colored numbers.
Green

Player bet on Black $ 1.-

Roulette Result : Black

Example 2 :

Player bet on Red $ 1.-

Roulette Result : Red

Hi Lo

Introduction
Hi Lo is as simple card game of guessing the next deal card is Hi, Lo , Same as well as cerrtain propotion bets. The game is single player game ,


How to Play

1. Player to select the start card manually or automatically by system
2. Once start card selected, player to place bet.
3. Select the Next deal card Hi, Lo, Hi or same , Low or Same or Same
4. when the next deal card result is ;
Correct - Win - the game continue with higher multiplier to win
Incorrect - Lose - the game ended. Player to continue with new round.
5. When player keep continue guess correctly, player can decide end the game by cashing out the winning amount.

Snow Sweeper

Description
Snow sweeper is classic game of minesweeper that is well-known. The goal of this game is to go through minefield, and avoid the mines. The longer player stay in the game the higher the reward will be!


Game Details

Player can choose Mine Field Map to play.

Game Symbol
Penguin Bonus Penguin Bomb

Each time player able to avoid mines, penguin will appear and player will be rewarded with multiplier.
When player hit the mines, Bomb will explode and player will lose the bet and any previous winnings.

How to Play

* Choose mine field Map for the number of mines
* Select number of Bomb in a map.
* Press "Start"
* Select field, avoid mines to be rewarded
* Collect anytime to win the game.

Disconnections

All unfinished rounds will be terminated every 20 minutes.
The unfinished round will be automatically collect after 20 minutes and the win will be credited to player's balance.
